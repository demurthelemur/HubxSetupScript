# HubxSetupScript

### A Bash script to install everything needed for React Native / Expo Development

###### Installation List:

- XCode Command Line Tools
- Homebrew
- XCode
- RVM & Ruby 3.0.0
- NVM
- NodeJS
- VSCode
- Slack
- Figma
- Notion
- Android Studio

The script also configures git email and username for you
